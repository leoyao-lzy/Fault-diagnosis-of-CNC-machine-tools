###   中文摘要
数控机床涉及多个学科技术,设备故障原因非常复杂。而故障诊断和排除的效率对企业生产效益有很大影响，单纯依靠维修工人的经验进行故障排除已经不能满足企业生产需求，因此故障诊断专家系统的研发具有重要意义。传统的故障诊断系统的算法有诸多弊端，而基于案例推理技术的专家诊断系统具有明显的技术优势。

本课题基于故障树分析法（FTA），通过SimRank算法建立了NL502SC数控机床故障诊断专家系统。前期调研了数控机床行业概况以及国内外差距；针对NL502SC数控机床，绘制了该机床各系统的故障树；对该机床故障现象进行分类，实现了“故障案例-故障现象”矩阵的构建；编写了一套C++语言程序，实现与待诊断案例相似度较高的几个历史故障案例的输出；编写了Python语言程序和MATLAB程序，完成了图形用户界面（GUI）的构建，给出了两种分别基于Python和MATLAB的NL502SC数控机床故障诊断可视化专家系统。

关键词：数控机床；故障诊断；故障树；图形用户界面



### ABSTRACT
CNC machine tools involve multiple disciplines of technology, and the causes of equipment failure are very complex. The efficiency of fault diagnosis and elimination has a great impact on the production efficiency of enterprises, and Relying solely on the experience of maintenance workers for troubleshooting can no longer meet the production needs of enterprises，so the research and development of fault diagnosis expert systems is of great significance. The algorithm of the traditional fault diagnosis system has many drawbacks, and the expert diagnosis system based on case inference technology has obvious technical advantages.

Based on the fault tree analysis method (FTA), this project established the NL502SC CNC machine fault diagnosis expert system through the SimRank algorithm. In the early stage, we investigated the overview of the CNC machine tool industry and the gap between home and abroad; for the NL502SC CNC machine, we have mapped the fault tree of each system of the machine; we classify the machine failure phenomena and realize the construction of the "fault case - fault phenomenon" matrix; We wrote a set of C++ language programs to implement the output of several historical failure cases that are similar to the cases to be diagnosed; We wrote the Python language program and the MATLAB program, completed the construction of the graphical user interface (GUI), and gave two NL502SC CNC machine fault diagnosis visualization expert systems based on Python and MATLAB, respectively.

Keywords: CNC machine tools; troubleshooting; fault trees; graphical user interface(GUI)
